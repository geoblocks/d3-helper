Here only the BaseD3ChartSVG class is used to draw and manage the caneva.

Then the example can focus only on the drawing of the pie-chart.

Note on pie-charts:

Pie-charts are not made to show both positives and negatives values. But if you have both, you can exclude one, draw them with another style (colors, borders) or draw an inner ring for negative values (an inner pie-chart). All solutions can be acceptable.
