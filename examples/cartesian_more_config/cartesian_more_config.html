The CartesianChart class reads the configuration to display the axis and the title.

The example draw horizontal bars in this cartesian chart.

Too long names for y-axis are automatically wrapped and ev. cut.

First example with a cartesian chart with, from the given configuration:

  • Labels for axis
  • A defined color
  • A title
  • A specific chart path.

Second example with a cartesian chart with the following supplementary configuration:

  • A d3 tick format to have pretty labels for the x axis.
  • A custom margin for left axis: more space for labels.
  • For Y axis, a key to get values, and another to get the displayed value. Typically for chart that superposes elements that have a same values. Here it displays the company name but identify them by the id. See the difference with the first chart; in the first example one line is not visible because they are at the same place.
